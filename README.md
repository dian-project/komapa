# KOMAPA - Koperasi Mahasiswa Pamulang

KOMAPA adalah aplikasi berbasis web yang dikembangkan untuk membantu pengelolaan
transaksi simpan dan pinjam pada Koperasi Mahasiswa Pamulang secara mudah,
terstruktur, dan efisien.

Aplikasi ini dirancang dengan tampilan modern menyerupai website e-commerce
serta menerapkan konsep responsive design sehingga dapat digunakan dengan
baik pada perangkat desktop maupun mobile.

## Fitur Utama
- Halaman Beranda dengan panduan penggunaan aplikasi
- Transaksi simpan dan pinjam anggota
- Daftar transaksi dalam bentuk tabel
- Halaman profil aplikasi
- Halaman login dan register
- Validasi form menggunakan JavaScript
- Desain responsif menggunakan media query CSS

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript
- GitHub Pages (Hosting)

## Tujuan Pengembangan
Aplikasi ini dibuat sebagai tugas mata kuliah Pemrograman Web serta sebagai
simulasi sistem informasi simpan pinjam koperasi mahasiswa.
